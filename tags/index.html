# 什么是PubSubJS？

PubSubJS 是一个用 JavaScript 编写的基于主题的 发布/订阅库。 PubSubJS 有同步解耦，所以主题是异步发布的。这有助于使您的程序保持可预测性，因为在消费者处理主题时，主题的发起者不会被阻止。

# Git使用手册

# 安装

官网下载：https://git-scm.com/downloads 下载完成后使用默认进行安装。

安装完成后，在开始菜单里找到 Git -> Git Bash，蹦出一个类似命令行窗口的东西，就说明Git安装成功！ 还需要最后一步设置，在命令行输入：

git config --global user.name "Your Name"
git config --global user.email "email@example.com"

2

因为Git是分布式版本控制系统，所以，每个机器都必须自报家门：你的名字和Email地址。
